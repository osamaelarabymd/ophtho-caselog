How to Use EyeLog

Everything you need to know in one place

🚀 Quick Start

1

Set up your profile

Go to Profile tab → Enter your name, PGY year, and institution

2

Log a case

Tap ➕ Log Case → Fill in procedure, role, date, attending → Save

3

Track your progress

Dashboard shows ACGME progress, charts, and streaks automatically

4

Export a report

Log Case tab → 📄 Export PDF or 📅 Monthly Report

✨ Features

⚡ Templates

Save common cases as templates for one-tap logging

🔄 Duplicate

Copy a previous case with today's date instantly

🔍 Search & Filter

Find cases by procedure, role, date, or keyword

📊 Analytics

Monthly trends, role breakdown, busiest days

🔥 Streaks

Track consecutive days of logging cases

📱 Mobile App

Add to iPhone home screen for native app feel

❓ FAQ

How do I add EyeLog to my iPhone?

Open the app in Safari → Tap Share button → Add to Home Screen → Add

Can I edit a case after saving?

Yes! Go to Case List → tap ✏️ on any case to edit all fields

How do I create a PDF for my PD?

Go to Log Case tab → tap 📄 Export PDF or 📅 Monthly Report

Is my data private?

Yes — only you and your program director can see your cases

How do I enable daily reminders?

Dashboard → tap 🔔 Reminders → allow notifications → get reminded at 6 PM daily

How do I use templates?

Log Case → fill form → tap + Save Template. Next time tap the template button to auto-fill
